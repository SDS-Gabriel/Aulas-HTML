# Aulas-HTML 1-7

🧠 Aprendizados Práticos e Recursos Implementados
Durante o desenvolvimento desta atividade prática, foram abordados e implementados os seguintes tópicos:

🛠️ Estruturação em HTML5:
Tags Semânticas: Uso correto de elementos como <header>, <main>, <section>, <article> e <footer> para melhorar a acessibilidade e o SEO.

Blocos de Conteúdo: Manipulação de títulos (<h1> a <h6>), parágrafos (<p>) e divisores textuais.

Hiperlinks e Navegação: Criação de menus e links utilizando as tags <nav>, <ul>, <li> e <a>.

Vinculação de Arquivos: Conexão correta entre o arquivo HTML e a folha de estilo externa através da tag <link> dentro do <head>.

🎨 Estilização em CSS3:
Seletores e Propriedades: Aplicação de regras usando seletores de tag, classe (.classe) e identificador (#id).

Modelo de Caixa (Box Model): Compreensão prática e ajuste de margin (margem externa), padding (espaçamento interno), border (bordas) e width/height (dimensões).

Tipografia e Cores: Customização de famílias de fontes (font-family), tamanhos (font-size), alinhamentos e definição de uma paleta de cores harmoniosa.

Layout e Alinhamento: Técnicas básicas de posicionamento de elementos e centralização de blocos na tela.
